## quick-git-tool
常用的git操作集合

#### 便捷食用说明

```
 npm install quick-git-tool -g
```

#### 1. quick-push  
``` bash  
quick-push -m "commit信息"
```

#### 2. quick-rebase  
``` bash  
quick-rebase 目标分支
```

#### 3. quick-set-gituser  
``` bash  

quick-set-gituser
username: your username
email: your email
username is set successfullyQAQ
email is set successfullyQAQ

```
