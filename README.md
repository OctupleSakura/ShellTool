### ShellTool
一些好用的shell脚本,比如常用的git操作集合等

### 目前有的脚本

#### 1. push  
``` bash  
push -m "commit信息"
```

#### 2. rebase  
``` bash  
quick_rebase 目标分支
```
#### 3. stash  
``` bash  
stash 目标分支
```

后续再继续补充啦

windows用户的话可以用**npm link**命令,将脚本映射到全局中并全局安装需要的模块
